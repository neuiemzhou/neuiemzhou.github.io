用hugo建站建好了之后怎么更新文章

  1. 首先新建一个md在这个里面写 hugo new posts/XXXX.md
  2. hugo server -t m10c –buildDrafts 在站点上查看更新的文章是否上去
  3. 根目录下 hugo –theme=m10c –baseUrl="https://neuiemzhou.github.io/”
  4. cd public
  5. git init(可省略)
  6. git remote add origin https://github.com/neuiemzhou/neuiemzhou.github.io.git（可省略）
  7. （只需要进行这个就可更新文章如果出错再加上前面的命令）进行git命令
  • git add .
  • git commit -m “随便什么都可以”
  • git push -u origin master
  • 输入github账号密码
  • 完事啦！！！

如果出错

  • 参考这篇文章：https://blog.csdn.net/qq_36079972/article/details/100284350

如果再出错

  • 去掉文章（.md）编写中的 ？=true
